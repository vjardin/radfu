# Legal Information

## Implementation of Public Specifications

RADFU implements the Renesas RA boot firmware protocol based on publicly available
documentation from Renesas Electronics:

- R01AN5562 - Standard Boot Firmware for RA Family MCUs (Cortex-M33)
  https://www.renesas.com/en/document/apn/standard-boot-firmware-ra-family-mcus-based-arm-cortex-m33

- R01AN5372 - System Specifications for Standard Boot Firmware (Cortex-M4/M23)
  https://www.renesas.com/en/document/apn/renesas-ra-family-system-specifications-standard-boot-firmware

These application notes are freely available from the Renesas website and describe
the UART/USB command protocol for device firmware updates.

No proprietary source code, firmware, or confidential material is included in this
repository.
